## For local development

- for new virtual environment:

    ```bash
    python3 -m venv ./venv
    ```

- activate virtual environment:

    ```bash
    source venv/bin/activate
    ```

- install dependencies:

    ```bash
    pip install -r requirements.txt
    ```

- run solution:

    ```bash
    python3 file.py
    ```
